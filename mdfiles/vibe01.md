# Introduction to Vibe Coding
Vibe coding is a modern approach to software development where the focus shifts from writing manual syntax to managing high-level intent using AI. Instead of typing every line of code, developers use natural language to describe what they want, and AI agents handle the technical implementation.

## The Evolution of Programming
Programming has become increasingly accessible over time:

Low-Level: Writing in assembly or machine code (complex, hardware-focused).

High-Level: Languages like Python and Java (human-readable but strict syntax).

No-Code/Low-Code: Building via visual blocks and drag-and-drop interfaces.

AI-Assisted: Tools like GitHub Copilot suggesting snippets of code.

Vibe Coding: Acting as a "conductor" for AI agents (Cursor, Windsurf, Replit Agent) that build and debug entire apps based on your instructions.

## Core Principles
1. Intent Over Syntax
The most critical skill is no longer memorizing commands, but articulation. You must clearly describe your app's logic, user flow, and "vibe" (the aesthetic and functional feel) so the AI understands your goal.

2. The Iterative Loop
Vibe coding is a conversation. You provide a prompt, review the AI's output, and refine the instructions until the product matches your vision.

Prompt: "Create a Streamlit flashcard app for linguistics."

Review: Test the generated app.

Refine: "Add a shuffle button and move the settings to the sidebar."

3. High-Level Verification
While you don't write the code, you must be able to read and verify it. A vibe coder ensures the AI's logic is sound and that the final product is secure and functional.
